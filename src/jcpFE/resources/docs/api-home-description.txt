<b>JCP APIs</b> reference and testing service.

<h2>JOSP APIs</h2>
<p>
    The <b>JOSP APIs</b> provide all method required by JOD and JSL agents to
    interact with the JOSP Cloud Platform. Objects and services can query this
    APIs to register them self and connect to the JCP Gateways. Although, with
    this API, objects can register events and statuses history.
</p>
<p>
    This APIs are provided by a JCP Service that provide also a
    <a href="/swagger-ui.html?urls.primaryName=JCP">JCP</a> Service's Status
    methods set.
</p>
<ul>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=Object">
      JOSP Objs
    </a>
    : Object's main methods group.
  </li>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=Service">
      JOSP Srvs
    </a>
    : Service's main methods group.
  </li>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=User">
      JOSP Usrs
    </a>
    : user's related queries from both objects and services.
  </li>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=Permissions">
      JOSP Permissions
    </a>
    : permissions manager APIs.
  </li>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=Gateways">
      JOSP GWs
    </a>
    : sets of methods for objects and services JOSP Gateways connection, and
    for JCP GWs registrations.
  </li>
  <li>
    <a href="/swagger-ui.html?urls.primaryName=Events">
      JOSP Events
    </a>
    : methods for object's events registration.
  </li>
</ul>
